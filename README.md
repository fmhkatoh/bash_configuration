# 4. わざわざ公開するということは普通のと違うの?

この設定ファイルの価値は、.bash_profileに書いたカラーコードをひたすら.bashrcのPS1やPS2で使っているというだけです。
早い話が、「ちょいとがんばれば大阪のおばちゃん系コンソールになるぞ」ということをやってみたかったのです。

まぁ、実際の話、バックグラウンドが黒だから、どんな色乗せても3色を越えると、コンソールの秘めたるオバハンパワーが開放されるので、そこは仕方ないね。
